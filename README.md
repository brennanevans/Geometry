# Geometry

Java application which can be used to create any simple polygon as well as perform geometrical calculations using them.

# Week4
This branch contains all the content required to create for Software2 Week4 Lab Exercises and Additional Exercises.

## Calculations Included:
- Area
- Perimeter
- Centroid
- Centroid of the vertex set (different to Centroid for non-triangular polygons)
- Whether it contains a point (either inside shape or as part of an edge)
- Distance of a point to the shape (either to the nearest edge or 0 if inside and shape is filled)
